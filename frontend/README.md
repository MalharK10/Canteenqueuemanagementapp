
  # Canteen Queue Management App

  This is a code bundle for Canteen Queue Management App. The original project is available at https://www.figma.com/design/sGJ7jwi9iGMHZwMEztik6q/Canteen-Queue-Management-App.

  ## Running the code

  Run `npm i` to install the dependencies.

  Set `VITE_API_BASE_URL` in `.env` to the backend origin, for example `http://localhost:3000`.

  Run `npm run dev` to start the development server.
